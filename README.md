# AI-Based Prediction of a Mass-Spring System Dynamics

This project was developed as part of a predictive modeling challenge, combining classical mechanics and artificial intelligence.

The objective is to simulate the behavior of a damped mass-spring system and train a neural network capable of forecasting its future position based on previous observations. The project demonstrates how machine learning can learn dynamic patterns from physical systems and perform short-term predictions.

The final visualization compares:

The actual motion generated by the physical model.
The position predicted by the neural network.
The prediction error throughout the simulation.

By integrating physics-based simulation with time-series forecasting, this project explores the intersection between engineering, data science, and artificial intelligence.
